# Nexli — Website

> Built by **Yashveer Singh** — Yashveer Labs

The marketing and landing site for the Nexli platform. Fast, clean, and conversion-focused.

## Stack

- **Framework**: Next.js with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
