A student code of conduct sets out the behavioral expectations for students, the rules they must follow, and the consequences for violations. It is the basis on which disciplinary action is taken. Without a written, communicated code, disciplinary actions can be challenged as arbitrary.

## Why a Written Code of Conduct Is Essential

**Fairness:** Students cannot be held to standards they were not informed of. The code must be communicated before it can be enforced.

**Consistency:** Written rules applied consistently prevent allegations of favoritism or discrimination.

**Legal protection:** Disciplinary actions based on a written, communicated code are more defensible in disputes.

**Parental partnership:** Parents who have read and acknowledged the code of conduct are partners in enforcing it.

## What the Code Must Cover

### Expected Conduct

Describe positive expectations, not just prohibitions:
- Respect for classmates, teachers, and staff
- Academic integrity (no cheating, plagiarism)
- Care for school property
- Punctuality and attendance
- Appropriate uniform and appearance

### Prohibited Conduct

Be specific about what is not allowed:
- Physical violence or threats of violence
- Bullying, harassment, or discrimination
- Theft or vandalism
- Possession of prohibited items (weapons, alcohol, tobacco, drugs)
- Cheating or academic dishonesty
- Disrespectful or abusive language toward staff
- Misuse of technology (unauthorized device use, cyberbullying)

### RTE Act Constraints

Under RTE Act 2009 and CBSE guidelines, the following are prohibited regardless of what the school's code says:
- Physical punishment of any kind
- Mental harassment
- Corporal punishment
- Detention or expulsion from Classes 1-8 on any grounds

Schools must ensure their disciplinary options within Classes 1-8 comply with these prohibitions.

### Disciplinary Procedure

Specify how disciplinary action works:
1. **Minor violations:** Teacher warning (verbal); noted in class register
2. **Repeated or moderate violations:** Written note home; parental meeting; counselor involvement
3. **Serious violations:** Formal written warning to parent; meeting with Principal; documented action plan
4. **Very serious violations:** Suspension; Parent-Principal-student meeting; conditions for return
5. **Extreme violations (secondary classes):** Expulsion process (with governing board involvement; following fair process)

### Due Process

Before any serious disciplinary action:
- Student has the right to explain their side
- Evidence is reviewed fairly
- Action is proportionate to the violation
- Parent is informed before action is taken (not after)

## Communicating the Code

- Included in the parent/student handbook given at admission
- Read and signed by parents and students (age appropriate)
- Reviewed at the start of each academic year (assembly, class discussion)
- Available on the school website

## FAQ

**Q: Can a school suspend a student in Classes 1-8 under RTE Act?**

A: RTE Act prohibits detention and expulsion but is less clear on suspension. Short-term suspension (1-2 days) is generally interpreted as within the school's authority even for elementary classes, provided it does not prevent the student from eventually completing their education. Prolonged suspension of elementary students is legally risky.

**Q: Must parents sign the code of conduct?**

A: Yes. Parent and student acknowledgment of the code of conduct at enrollment (and renewal each year) is strongly recommended. It confirms the rules were communicated.

**Q: Can a student be expelled for a single serious incident?**

A: For very serious incidents (violence causing injury, possession of weapons, drug distribution), expulsion may be proportionate even for a first incident. The process must still be fair: the student heard, evidence reviewed, action documented. For secondary students, this may be legally defensible; for elementary students, the RTE Act prohibition on expulsion applies.

**Q: What if a student challenges a disciplinary action as unfair?**

A: The school's grievance procedure should cover student appeals. If the code of conduct was followed, the action was proportionate, and due process was observed, the school is in a defensible position. Documentation of each step is essential.
