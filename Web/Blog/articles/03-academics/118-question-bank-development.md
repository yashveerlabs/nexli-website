## Executive Summary

A centralized question bank saves 25% of assessment creation time and ensures consistent quality. Schools using organized question banks report faster test creation and more balanced assessments across sections.

---

## Why a Question Bank?

**Benefits:**
- Reuse quality questions across years
- Ensure balanced assessment (mix difficulty levels)
- Faster test creation (select instead of write)
- Consistency across sections (same questions, different students)
- Continuous improvement (refine questions based on performance)

---

## Structure: Organize by Multiple Dimensions

**Organize questions using:**

| Dimension | Examples |
|---|---|
| **Grade & Subject** | Class 10 Science; Class 9 English |
| **Topic/Chapter** | Chapter 3 - Chemical Reactions; Unit 2 - Poetry |
| **Learning Outcome** | "Identify chemical reaction types"; "Analyze poetic devices" |
| **Bloom Level** | Remember, Understand, Apply, Analyze, Evaluate, Create |
| **Difficulty** | Easy (E), Medium (M), Hard (H) |
| **Question Type** | MCQ, Short Answer, Long Answer, Numerical, Case Study, Project |

---

## Question Bank Development

### Step 1: Plan Coverage

Map all topics and learning outcomes for each subject/grade.

**Example:**
- Class 8 Math: 15 chapters
- Chapter 1 (Linear Equations): 10 learning outcomes
- Per outcome: Plan 5-10 questions

### Step 2: Write Questions

**For each learning outcome, write:**
- 3-5 Easy questions (Remember/Understand level)
- 3-5 Medium questions (Apply/Analyze level)
- 2-3 Hard questions (Evaluate/Create level)
- Mix formats: MCQ, short answer, application, problem-solving

### Step 3: Quality Review

**Check each question:**
- [ ] Clear, unambiguous wording
- [ ] Correct answer documented
- [ ] Aligns to specific learning outcome
- [ ] Appropriate difficulty for level
- [ ] No culturally insensitive content
- [ ] No trick questions (unless intentional)

### Step 4: Organize & Store

**Digital storage with tags:**
- Grade: 8
- Subject: Math
- Chapter: 3 (Linear Equations)
- Topic: Solving linear equations in one variable
- Learning Outcome: "Solve equations; verify solution"
- Bloom: Apply
- Difficulty: Medium
- Type: MCQ
- Correct Answer: B

### Step 5: Implement & Refine

**When creating assessments:**
- Search by topic and difficulty
- Select appropriate questions
- Assemble test/quiz

**After assessment:**
- Analyze performance (which questions were hard? Easy?)
- Update question notes ("90% students got this wrong, revise wording")
- Difficulty tag (if medium question performed as hard, update)

## Sample Question Bank Entry

```
QUESTION BANK ENTRY

Grade: 10 | Subject: Science | Chapter: Chemical Reactions
Topic: Types of reactions | Learning Outcome: Identify reaction types
Bloom: Understand | Difficulty: Easy | Type: Short Answer

Question:
"What type of reaction is: Magnesium + Oxygen → Magnesium Oxide?"

Answer Key:
"Combination reaction (synthesis reaction), two or more substances combine to form one product."

Difficulty Analysis:
- 2024: 85% students correct
- 2025: 78% students correct
- Notes: Slightly harder than expected; consider revising if <70% next year

Related Outcomes:
- Also addresses: Write and balance chemical equations
```

## Real-World Impact

- Test creation time: 2-3 hours per test → 30 minutes (select from bank)
- Assessment consistency: Varies widely → Standardized format and difficulty
- Teacher collaboration: Isolated work → Shared resource building
- Question quality: Variable → Continuously refined based on performance data

## Question Bank Checklist

- [ ] Map all topics and learning outcomes
- [ ] Write 8-12 questions per outcome (mix difficulties)
- [ ] Review quality of each question
- [ ] Organize with clear tagging system
- [ ] Store digitally with secure access
- [ ] Document correct answers and explanations
- [ ] Implement version control (track question changes)
- [ ] Analyze performance after use; refine questions
- [ ] Archive old questions (maintain for comparison)
- [ ] Train teachers on using the bank
