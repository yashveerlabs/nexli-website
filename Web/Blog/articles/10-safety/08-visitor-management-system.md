## Visitor Management System for Schools: Digital Security

**Controlling campus access is critical for student safety. A digital visitor management system automates the process: OTP verification, expected-visitor pre-registration, audit trails, and reports. This guide covers manual and digital approaches.**

---

## Manual System (Paper-Based)

**Visitor Log Book:**
- Gate keeper maintains physical register
- Entries: Name, purpose, time in, time out, phone, ID copied
- Staff verification: Gate keeper calls office to verify visitor
- Badge issued (color-coded or numbered)

**Limitations:** Inconsistent, hard to search, data loss risk, no real-time alerts.

---

## Digital System (Using Nexli or Similar)

**Expected Visitor Pre-Registration:**
- Parents register visit: "I want to visit my child's classroom on Friday 2–3 PM"
- System sends confirmation: "Visit approved. Show OTP at gate."
- Gate staff see expected visitor list daily

**OTP Verification:**
- Visitor arrives at gate, provides phone number
- System sends OTP (one-time password) to visitor's phone
- Visitor enters OTP to gate staff; staff verifies and grants entry
- Entry auto-logged with timestamp, visitor identity, purpose

**Audit Trail:**
- All entries logged (name, purpose, time in, time out, approver)
- Real-time alerts: Principal notified of any unexpected visitor
- Monthly report: How many visitors, which purposes, any repeats?

**Blacklist Management:**
- List of persons NOT allowed on campus (court order, safety concern)
- Gate staff check visitor name against blacklist before issuing OTP
- If on blacklist: Entry denied, Principal alerted

---

## Implementation Steps

1. **Choose system:** Nexli's Visitor Management module or third-party app
2. **Set policy:** Who is an "expected visitor"? What purposes?
3. **Train staff:** Gate keeper learns system; confirm daily list review
4. **Parent communication:** Explain system to parents (benefits: fast entry, security tracking)
5. **Monitor:** Weekly review of visitor patterns; adjust if needed

## FAQ

**Q1: Will OTP system slow down visitor entry?**  
A: No. If visitor is pre-registered, OTP takes <1 minute. Unexpected visitors may take 5–10 minutes (verification time).

**Q2: Should parents visiting class need approval?**  
A: Yes. Pre-approval (even if just a class teacher check) prevents unknown people on campus.

**Q3: What if a visitor refuses OTP?**  
A: Gate staff don't grant entry. It's a security requirement. Most visitors accept after explanation.
