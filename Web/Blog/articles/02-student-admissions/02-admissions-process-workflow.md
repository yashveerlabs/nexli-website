## Executive Summary

A well-designed admissions process converts inquiries into enrollments efficiently. This article walks through every step: from the first phone call to the student walking through the gate on day 1. Timeline: 6–10 weeks from inquiry to enrollment.

---

## End-to-End Admissions Workflow

### Phase 1: Lead Capture (Day 0–1)

**Step 1: Inquiry Intake**
- Parent calls, emails, or fills web form
- Receptionist answers warmly; collects name, phone, child's name, grade interested
- Inquiry logged into system
- Auto-confirmation SMS sent within 1 hour

**Example SMS:** "Thank you for inquiring about XYZ School! Our admissions team will be in touch within 24 hours."

**Timeline:** Same day

### Phase 2: Lead Qualification (Day 1–2)

**Step 2: Information Package & Initial Call**
- Admissions officer calls parent within 24 hours
- Discusses child's background, school preferences, expectations
- Sends information package (fee structure, admissions timeline, application form)
- Answers parent questions

**Call Script:**
- "Hi, this is Priya from XYZ School. I'm calling to learn more about your child and answer any questions about our school."
- "What grade is [child] currently in?"
- "What are you looking for in a school?"
- "When would you like them to start?"

**Timeline:** Day 1–2

**Output:** Parent receives info package; decides whether to proceed

### Phase 3: Application (Day 3–10)

**Step 3: Application Submission**
- Parent completes application form (online or paper)
- Required info: Child's name, DOB, previous school, medical info, parent contact
- Required documents: Previous school TC, Aadhaar photocopy, address proof, birth certificate

**Step 4: Document Verification**
- Coordinator reviews submitted documents
- Checks completeness: All docs present?
- Verifies authenticity: TC matches age, Aadhaar format correct
- If incomplete: Email parent requesting missing docs
- If complete: Move to testing/interview

**Timeline:** Day 3–10

**Output:** Application approved/rejected or documents requested

---

### Phase 4: Assessment (Day 10–20)

**Step 5: Entrance Test (If Applicable)**
- School conducts entrance exam to assess learning level
- Format: 60-min test (Math, English, Reasoning)
- Grading: Qualitative (not about pass/fail; about identifying learning level)
- Result: Shared with parents within 3 days

**Timing:**
- Test date announced in info package
- Parent confirms attendance 1 week before
- Reminder sent 3 days before

**Timeline:** Day 10–15

**Output:** Assessment score; used in interview

**Step 6: Interview**
- Conducted by Principal + Academic VP + sometimes Counselor
- Format: 20–30 min with parent + child
- Assessment: Academic level, learning style, behavior, fit with school
- Topics discussed: Why do you want our school? What are your interests? How do you handle challenges?

**Interview rating:** Excellent / Good / Average / Concerns

**Timeline:** Day 15–20

**Output:** Admission decision (Offer / Waitlist / Rejection)

---

### Phase 5: Offer (Day 20–25)

**Step 7: Offer Communication**
- Principal calls parent with decision within 3 days of interview
- Explains offer clearly: "We're pleased to offer admission for Class V starting April 1"
- Outlines next steps: Enrollment fee, parent agreement, orientation
- Answers questions

**If offered:** Enrollment process begins (Step 8)

**If waitlisted:** "We have limited seats. If a seat opens, we'll contact you by [date]. Would you like to remain on the list?"

**If rejected:** "Thank you for your interest. Unfortunately, we cannot offer a seat at this time. We encourage you to reapply next year. We can suggest other schools if helpful."

**Timeline:** Day 20–25

**Output:** Parent accepts or declines offer

---

### Phase 6: Enrollment (Day 25–40)

**Step 8: Enrollment Agreement**
- Parent signs enrollment agreement (school policies, fee commitment, etc.)
- Collects enrollment fee (₹5K–50K depending on school)
- Completes additional forms: Medical clearance, emergency contacts, transport, hostel (if applicable)

**Timeline:** Day 25–35

**Step 9: Document Submission**
- Parent submits required documents:
  - Aadhaar copy (for ID proof)
  - Medical fitness certificate
  - Immunization records
  - Previous school TC
  - Address proof
  - Caste certificate (if applicable)
  - Bank details (for refunds)

**Coordinator verifies:** All documents received, legible, authentic

**Timeline:** Day 30–35

**Step 10: Orientation & Onboarding**
- School schedules parent-child orientation
- Tour of campus (library, lab, sports facility, classrooms)
- Meet teachers and counselor
- Explain daily schedule, rules, expectations
- Address any concerns

**Timeline:** Day 35–40

**Step 11: Final Confirmation**
- Coordinator confirms enrollment in system
- Student photo taken and ID created
- Parent portal activated (parent can see child's timetable, homework, announcements)
- Uniform and supplies list emailed
- First day date and time confirmed

**Timeline:** Day 40

**Output:** Student enrolled; ready to start

## Parallel Admissions Cohorts

**Reality:** Schools run multiple admissions simultaneously

**Cohort 1 (January–March):** Classes 1–5, 6–8, 9
**Cohort 2 (May–June):** Class 11
**Cohort 3 (Ongoing):** Mid-year admissions (Class 2–10)

**Each cohort has its own timeline; they run in parallel.**

## Key Decision Points

### 1. Application Approval/Rejection

**Criteria:**
- Complete documents?
- Age within range?
- Previous school in good standing?
- Special needs (if any) that school can support?

**Authority:** Admissions officer approves; Principal escalates if unsure

### 2. Interview Rating Conversion to Offer/Waitlist/Rejection

**Criteria (example):**
- Excellent: Offer
- Good: Offer (if multiple seats available) OR Waitlist (if limited seats)
- Average: Waitlist
- Concerns: Rejection (or offer with additional support)

**Authority:** Principal decides

### 3. Waitlist → Offer Conversion

**When does waitlist convert?**
- When an accepted student declines enrollment
- Timeline: During summer (June–July) before school starts

**Fairness:** Waitlist is prioritized by interview date (first interviewed, first offered)

## Common Admissions Process Failures

### Failure 1: No Documented Timeline
"When should docs be requested?" → No answer. Things drag on.

**Solution:** Document timeline. Post it. Follow it.

### Failure 2: Lost Applications
"Did we receive your application?" → Unclear. Applicant gets frustrated.

**Solution:** System logs all applications; auto-confirmation sent.

### Failure 3: Slow Decision Communication
"When will I know the result?" → 2 weeks later. Family has enrolled elsewhere.

**Solution:** Communicate timeline upfront. Make decisions within promised timeline.

### Failure 4: Inconsistent Assessment
"Why was one child accepted and another rejected with similar scores?"

**Solution:** Document assessment criteria. Use rubric. Consistency check.

## FAQ

**Q: How long should admissions take?**
A: 6–8 weeks is typical. Faster: 4–6 weeks (if you're organized). Slower: 10+ weeks (inefficient).

**Q: Should we do rolling admissions or cohort-based?**
A: Most schools use cohort-based for Classes 1–10 (March deadline) and rolling for mid-year. This predictability helps.

**Q: What if a family wants instant admission?**
A: Mid-year admissions can be faster (4 weeks if documents are ready). But summer admissions for new classes require timeline.

**Q: Should entrance test be mandatory?**
A: Most schools require it to assess learning level. Waive for kindergarten. Mandatory for Class 1+.

**Q: How do we handle multiple children from same family?**
A: Discount on enrollment fee (sometimes). Each child goes through full admissions process; decisions made individually.
