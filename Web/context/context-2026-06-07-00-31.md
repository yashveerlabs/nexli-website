## Architecture decisions (locked)
- React + Vite **client SPA / installable PWA**; Capacitor for native later.
- Styling: **Tailwind v4 + NEXLI tokens + ported reference component CSS** (nexli.css). Use bespoke `.kpi/.panel/.donut/...` classes + Tailwind utilities.
- **Free-tier first (Spark, no Blaze).** No Cloud Functions deployed — clean seams + local Admin scripts later.
- **RBAC via protected Firestore profile docs** (`/userIndex/{uid}`, `/schools/{id}/members/{uid}`) enforced in security rules with `get()`. Custom claims = later perf optimization. Multi-tenant isolation by `schoolId` at rules layer.
- Auth: email/pass (staff) + **phone OTP (parents, no password)**; no SMS MFA (kept MFA-ready).
- **Students/parents cannot change own passwords**; only `user.password.manage` roles (Coordinator, CS HOD, School Admin, Principal, IT Admin).
- **Major data-entry forms = dedicated routed pages**, never modals (modals only for confirm/preview/warn/simple-update).
- **Payments:** no gateway — school QR images + bank details, parents pay offline, staff record + receipt. Gateway seam preserved.
- **Notifications:** in-app only (+ announcements/notice board/parent comms); info cards for future WhatsApp/SMS/email/FCM.
- **Images:** ImageKit (profile photos + QR) as URLs; initials-on-gradient fallback until ImageKit keys + free signing endpoint provided.
- **Maps/GPS:** Leaflet + OpenStreetMap + driver-PWA geolocation → Firestore realtime; seam for Google Maps/GPS HW.
- **AI:** build every AI surface fully but NO provider — glossy "coming soon" overlay (`AILockedOverlay`) behind `aiEnabled` flag.

## Design system decisions
Obsidian/Gold/Ivory. Tokens in nexli.css `:root` (--bg #080808, --card #181818, --gold #C6A55C, --text #F7F2E8, --ease cubic-bezier(0.22,1,0.36,1)). Inter font. Mobile-first inversion of the (desktop-led) reference: tables→cards on mobile, bottom-nav + drawer, no horizontal scroll, 60fps low-end, WCAG AA.
